# Change Your Name
Use the `!setname <YOURNAME>` command to change your name. Replace the placeholder `<YOURNAME>` with a name that should be used when the TTS announces who speaks.
There is a limitation of a maximum of 25 characters for the name.

# Change Your voice
Use the `!setvoice <YOURVOICE>` command to change the voice the TTS is using when speaking your sentences. Replace the placeholder `<YOURVOICE>` with one of the following available voices, either by number or by full name.
